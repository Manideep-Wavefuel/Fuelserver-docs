Item Type

Item Type define common properties and functionalities that multiple items of the same type share. For example, an "Item Type" could be "Light Bulb," which would define properties like brightness, color temperature, and on/off state.

This documents helps you to understand and implement ItemType. These APIs are specialized tools that help you to perform certain operations on ItemType.

With the Help of these APIs you can create, read, update, and delete ItemType. The below is the API documentation which contains API url paths, message structure and the description on what the API does. If there is any issue, you can refer to the Errors section to get good insights on the error.

1. Create Itemtype

This API is used to create a ItemType. It takes the below mentioned API parameters as input and returns the newly created ItemType and its aws thing type in response.
:organizationId should be replaced with the Id of the Organization.

POST /api/:organizationId/itemType/create/one → {ItemTypeAndThingTypeResult}


Name Type Attribute Description
name string The name of the itemtype. Must be a non-empty string.
vendorId string The vendor ID associated with the itemtype. Must be a non-empty string.
description string optional (Optional) Description of the itemType.
parameterIds Array.<string> optional (Optional) Parameter IDs associated with the itemType.

2. Create Many Item types

This API is used to create multiple ItemTypes. It takes the below mentioned API parameters as input and returns the newly created ItemTypes and their AWS thing types in the response.
:organizationId should be replaced with the Id of the Organization.

POST /api/:organizationId/itemType/create/many → {Array.<ItemTypeAndThingTypeResult>}


Name Type Attribute Description
data Array.<object> The array of itemtype input object
data[].name string The name of the itemType. Must be a non-empty string.
data[].vendorId string The vendor ID associated with the itemtype. Must be a non-empty string
data[].description string optional (Optional) Description of the itemType.
data[].parameterIds Array.<string> optional (Optional) Parameter IDs associated with the itemType.

3. Read Item Type

This API is used to read a ItemType. It takes the parameter Id as input in the url and returns the requested ItemType in response.
:organizationId should be replaced with the Id of the Organization.
:itemTypeId should be replaced with the Id of the ItemType.

GET /api/:organizationId/itemType/read/one/:itemTypeId → {ItemType}


4. Read Many Itemtypes

This API is used to read multiple ItemTypes. It takes below parameters as input and return the ItemTypes in response.
:organizationId should be replaced with the Id of the Organization.

POST /api/:organizationId/itemType/read/many → {Array.<ItemType>}


Name Type Attribute Description
where Object The condition object for querying records with itemtype fields.
orderBy object optional Use orderBy to sort a list of records or a nested list of records by a particular field or set of fields.
select object optional Use select to return specific fields by assigning true to it.
skip number optional uses skip to skip a certain number of results
take number optional uses take to select a limited range

5. Update Item Type

This API is used to update a ItemType and return the updated ItemType in the response.
:organizationId should be replaced with the Id of the Organization.
:itemTypeId should be replaced with the Id of the ItemType.

PATCH /api/:organizationId/itemType/update/one/:itemTypeId → {ItemType}


Name Type Attribute Description
name string optional The name of the itemtype must be a non-empty string
description string optional (Optional) Description of the itemType.
parameterIds Array.<string> optional ((Optional) Itemtype IDs associated with the itemType.

6. Update Many Item Types

This API is used to update a ItemTypes and return the updated ItemTypes in the response.
:organizationId should be replaced with the Id of the Organization.

PATCH /api/:organizationId/itemType/update/many → {Array.<ItemType>}


Name Type Attribute Description
data Array.<object> The array of itemType input object.
data[].id string The id of the itemtype. Must be a non-empty string
data[].name string (Optional) The name of the Itemtype. Must be a non-empty string.
data[].description string optional (Optional) Description of the itemType.
data[].parameterIds Array.<string> optional (Optional) Parameter IDs associated with the itemType.

7. Delete ItemType

This API is used to delete a ItemType and return the deleted ItemType in response.
:itemTypeId should be replaced with the Id of the ItemType.
:organizationId should be replaced with the Id of the Organization.

DELETE /api/:organizationId/itemType/delete/one/:itemTypeId → {ItemType}


8. Delete Many Item Types

This API is used to delete multiple ItemTypes and return the deleted ItemTypes in response.
:organizationId should be replaced with the Id of the Organization.

DELETE /api/:organizationId/itemType/delete/many → {ItemType}


Name Type Description
ids Array.<string> Array of ItemType Ids to be deleted.
